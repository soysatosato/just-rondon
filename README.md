# Just Rondon

A personal full-stack web application built with **Next.js**, **Supabase**, and **Vercel**.

## Overview

Just Rondon is a responsive web app that allows users to explore museums, musicals, artworks, and post comments. The project demonstrates full-stack development skills including authentication, database integration, server-side logic, and email notifications.

## Features

- User authentication with Clerk
- CRUD operations for museums, artworks, and musical content
- Comment board for user interaction
- Responsive design for desktop and mobile
- Email notifications via Nodemailer
- Interactive maps using Leaflet
- Carousels and animations with Framer Motion and Embla Carousel

## Tech Stack

- **Frontend:** Next.js 14, TypeScript, Tailwind CSS
- **State Management:** Zustand
- **Backend / Database:** Supabase (PostgreSQL, Auth, Storage)
- **Email / Notifications:** Nodemailer
- **Hosting / Deployment:** Vercel
- **Other Libraries:** React Leaflet, React Markdown, Remark GFM

## Local Setup

```bash
# Clone the repository
git clone https://github.com/soysatosato/just-rondon.git
cd just-rondon

# Install dependencies
npm install

# Run development server
npm run dev
```
